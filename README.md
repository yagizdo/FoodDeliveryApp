<br/>

## Run Locally

Clone the project

```bash
  git clone https://github.com/yagizdo/FoodDeliveryApp
```

Open the project file

```bash
  open FoodDeliveryApp.xcodeproj
```

Run App

```bash
  run in XCode
```
